# String Lib

## The ```string``` library

```py
import string
```

Includes these utilities:

* ```string.split()```, cuts the string on every whitespace and returns an array of all the parts
* ```string.replace(string, target, replacement)```, tries to find the target in the given string and replaces it with its given replacement
* ```string.trim()```, removes all whitespaces
* ```string.toUpper()```, returns the string in uppercase
* ```string.toLower()```, returns the string in lowercase
